# Note Painting

Note Painting (NP) to język operacji wizualnych.  
Obraz nie jest tu rysowany ręcznie — jest **wynikiem wykonania programu**.  
Program składa się z jednostek, relacji, transformacji i pól, które tworzą strukturę końcową.

Repozytorium dokumentuje **system NP** w formie języka formalnego.

---

## Struktura systemu

System NP składa się z pięciu warstw:

1. **Units**  
   Podstawowe elementy języka: klucze, splity, linie, gradienty, punkty.

2. **Relations**  
   Zasady łączenia jednostek w zdania i sektory.

3. **Transformations**  
   Operacje zmieniające przestrzeń, stan i geometrię.

4. **Fields**  
   Struktura pól powstająca przez podziały — przestrzeń działania języka.

5. **Structures**  
   Wynik końcowy: obraz jako struktura logiczna.

Każda warstwa jest opisana w folderze `system/`.

---

## Cel projektu

Celem repozytorium jest:

- udokumentowanie języka NP w sposób formalny,  
- przedstawienie jego logiki, struktury i zasad,  
- oddzielenie języka od implementacji,  
- stworzenie czytelnej mapy systemu dla odbiorców i badaczy.

To repozytorium nie zawiera kodu wykonawczego — jest **specyfikacją języka**.

---

## Minimalna idea

Note Painting to:

- **język**, nie styl,  
- **struktura**, nie dekoracja,  
- **operacja**, nie gest,  
- **deterministyczny proces**, nie interpretacja.

Obraz powstaje z logiki, nie z intuicji.
